The Real Trade-offs

  • Provisioned concurrency costs ~$15/month per instance: At 10 provisioned instances, that's $150/month for one function. A comparable Fargate task costs less with more flexibility.
  • SnapStart is the real innovation: For Java/JVM Lambdas, SnapStart reduces cold starts from 6s to 200ms by snapshotting initialized containers. No ongoing cost premium.
  • Most cold starts don't matter: For async workloads (SQS consumers, S3 triggers), a 1-second cold start is invisible. Only synchronous API endpoints with p99 SLAs need mitigation.
  • Runtime choice is the cheapest fix: Switching from Java to Node.js or Python cuts cold starts by 5-10x. The language choice has more impact than any AWS feature.
  • Application-level warmup is free: CloudWatch scheduled events pinging your Lambda every 5 minutes costs nothing and keeps 1 container warm. Not perfect, but free.
