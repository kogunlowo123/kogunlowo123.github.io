The Real Trade-offs

  • Vault's killer feature is dynamic secrets: Generating short-lived database credentials on-demand is something Secrets Manager can't do. If you need ephemeral credentials, Vault is unmatched.
  • Operational burden is real: Vault clusters need HA, auto-unseal, backup, monitoring, and upgrades. That's a full-time job. Secrets Manager is zero-ops.
  • Multi-cloud requires Vault: If you run workloads across AWS, Azure, and GCP, a cloud-native secrets manager locks you into one provider. Vault is cloud-agnostic.
  • Cost at scale: Secrets Manager charges $0.40/secret/month. At 10,000 secrets, that's $4,000/month. Vault on EC2 might be cheaper at scale but ops cost is hidden.
  • Integration depth: Secrets Manager integrates natively with RDS, Lambda, ECS, and EKS. Vault requires sidecar injectors and CSI drivers — more moving parts, more failure modes.
