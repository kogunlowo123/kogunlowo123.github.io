Cognito Is AWS's Worst Managed Service

Cognito's documentation is confusing, its hosted UI is ugly, migration is nearly impossible, and Lambda triggers for basic customization are a nightmare. Auth0 has a beautiful dashboard, pre-built UI components, and works in 15 minutes. Cognito takes 15 days of frustration.

The Real Trade-offs

  • Cognito is free for 50K monthly active users — Auth0 charges $23/month after 7.5K
  • Auth0 Actions replace Cognito's complicated Lambda trigger system with a simple Node.js editor
  • Entra ID is unbeatable for enterprise SSO, conditional access, and Active Directory integration
  • Cognito user pool migration requires re-authenticating every user — there's no bulk password import
  • If you're already on AWS and need basic auth, Cognito works — but expect to fight the documentation
